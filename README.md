# solar-dryer-iot-system

Final Year Project: Design of a Portable Solar Dryer for Agricultural Products

## Features
- ESP32 Wokwi simulation
- DHT22 temperature and humidity monitoring
- Soil moisture simulation using potentiometer
- PHP and MySQL backend
- Real-time web dashboard
- Drying session control
- Fan monitoring and warning alerts
- Crop drying guide

## Tools Used
- PHP
- MySQL
- XAMPP
- Wokwi
- ESP32
- ngrok
- JavaScript
- Chart.js

## Note
This project uses Wokwi simulation instead of physical hardware.
